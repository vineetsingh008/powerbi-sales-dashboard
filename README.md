# powerbi-sales-dashboard

## Project Overview

This Power BI dashboard analyzes retail sales performance using the Sample Superstore dataset. The dashboard provides insights into category performance, regional sales distribution, monthly sales trends, top-selling products, and profit contribution.

## Tools Used

* Power BI Desktop
* DAX
* Excel Dataset
* GitHub

## Dashboard Features

* Sales by Category
* Sales by Region
* Monthly Sales Trend
* Top Selling Products
* Profit by Category

## Key Insights

* Technology generated the highest sales.
* West region contributed the largest share of revenue.
* Monthly sales fluctuated throughout the year with strong performance in the final quarter.
* A small number of products contributed a significant portion of total sales.
* Technology and Office Supplies generated higher profits compared to Furniture.
